Energy Save Control - ASERFINTEC

Credenciales demo:
usuario: admin
clave: 1234

Cómo abrir:
1. Descomprimir el archivo ZIP.
2. Abrir la carpeta en Visual Studio Code.
3. Ejecutar el archivo index.html con Live Server.

Funciones incluidas:
- login demo
- branding ASERFINTEC / ASFT
- dashboard ejecutivo con KPIs
- gráficas estadísticas
- formulario de registros
- edición y eliminación
- filtros por categoría y responsable
- búsqueda libre
- exportación a CSV
- impresión de reporte
- cambio de tema oscuro / claro
- carga y restauración de datos demo

Nota:
La aplicación usa Chart.js por CDN para las gráficas. Si no ves las gráficas, verifica conexión a internet o abre la consola del navegador para comprobar bloqueo del CDN.
